# Cadastro de Alunos em Python

Projeto desenvolvido para a faculdade.

## Funcionalidades
- Cadastro de alunos
- Validação de nome, idade e nota
- Classificação (aprovado, recuperação, reprovado)
- Cálculo da média da turma
- Relatório com:
  - quantidade de alunos por situação
  - maior e menor nota

## Tecnologias
- Python
